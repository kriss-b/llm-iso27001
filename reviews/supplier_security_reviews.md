# Supplier Security Reviews

## 📌 Purpose
This document tracks the **security posture and compliance status** of third-party suppliers used by **ACME CORP**. It ensures that suppliers meet the organization's security requirements and align with **ISO 27001:2022** controls (A.15).

---

## 📋 Supplier Security Review Process
1. **Initial Vetting**: Assess supplier security practices during the selection process.
2. **Contractual Agreements**: Ensure contracts include security and compliance clauses.
3. **Ongoing Monitoring**: Review supplier security posture annually or after significant changes.
4. **Incident Reporting**: Require suppliers to report security incidents within 24 hours.
5. **Termination**: Define procedures for secure termination of supplier relationships.
